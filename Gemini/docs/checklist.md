# Checklista: Inför 'Koda som ett Ess'

## 30 minuter innan (15:30)
- [ ] Kontrollera internetuppkoppling.
- [ ] Kör `gemini login` och verifiera att du är inloggad.
- [ ] Kör `npm install` i mappen `demo-app`.
- [ ] Öppna VS Code och zooma in (viktigt för läsbarhet!).
- [ ] Kontrollera att `npm test` fungerar (det bör visa 0 tester eller ett felmeddelande, vilket är korrekt då vi inte skrivit testerna än).

## Vid start (16:00)
- [ ] Hälsa alla välkomna och visa `workshop-plan.md`.
- [ ] Säkerställ att alla har Node.js och Gemini CLI installerat.
- [ ] Be alla köra `gemini login` samtidigt för att få det avklarat.

## Under matpausen (17:00 - 17:30)
- [ ] Ladda datorn om det behövs.
- [ ] Svara på eventuella individuella frågor om installationer.

## Tips för live-demo
- Prata högt om vad du skriver ("Nu skriver jag en prompt som är specifik kring att jag vill ha interface...").
- Om Gemini ger ett oväntat svar – använd det! Det visar deltagarna hur man justerar sina prompts.
